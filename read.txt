# Attendance Management System

## Overview
The Attendance Management System is a web-based application designed to simplify the process of tracking student attendance in educational institutions. It provides features for administrators, teachers, and students to manage attendance records efficiently.

## Features
- **Admin Panel**: Allows administrators to manage classes, student enrollments, and user accounts.
- **Teacher Dashboard**: Provides teachers with tools to mark attendance, view class rosters, and generate attendance reports.
- **Student Dashboard**: Enables students to view their attendance records for various subjects and periods.
- **Attendance Tracking**: Allows teachers to mark attendance for students in real-time or based on predefined schedules.
- **Reports and Analytics**: Generates detailed attendance reports for classes, subjects, and individual students.
- **User Management**: Provides functionalities for user authentication, authorization, and profile management.

## Technologies Used
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT (JSON Web Tokens)

## Getting Started
1. Clone the repository: `git clone https://github.com/shaif-codes/attendance-management-system.git`
2. Install dependencies: `npm install`
3. Set up the backend:
   - Navigate to the `backend` directory: `cd backend`
   - Install backend dependencies: `npm install`
   - Start the backend server: `npm start`
4. Set up the frontend:
   - Navigate to the `frontend` directory: `cd frontend`
   - Install frontend dependencies: `npm install`
   - Start the React development server: `npm start`
5. Access the application in your browser at `http://localhost:3000`.
